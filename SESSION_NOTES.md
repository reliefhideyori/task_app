# タスク管理・効率化プロジェクト - セッションメモ

## 背景・課題

### 本人の状況
- **役割**: 保険代理店（本業）、青年会議所(JC)、Webマーケ事業、地域イベント等
- **ADHD傾向あり**: ワーキングメモリの弱さ、先延ばし癖
- **現在使っているツール**: Googleカレンダー、LINEリマインダ（たまに）
- **主な連絡手段**: LINE、電話（Gmailではない）
- **朝のルーティン**: なし。起きたらスマホを見る

### 核心的な問題
```
タスク発生 → メモすらしない → 忘れる → 溜まる → 現実逃避 → さらに溜まる
```

### 過去の失敗パターン
- リマインダアプリは最初うまくいったが徐々にやめた
- タスクが溜まる → アプリを開く＝苦痛 → 開かなくなる
- 「自分から見に行く（PULL型）」のツールはいずれ破綻する

---

## 設計方針（合意済み）

### 3つの原則
1. **PUSH型のみ**: 自分からアプリを開く必要をゼロにする
2. **入力ハードル最小化**: メモすらしないなら、メモしなくていい仕組み
3. **タスク非表示**: 一度に見えるのは今日やる3つだけ（現実逃避トリガー排除）

### 通知手段
- **Gmail**が一番見る（朝スマホで確認する習慣あり）

---

## 全体構想（議論中）

### システム全体像
1. **朝のブリーフィング** - 毎朝Gmailに今日の予定＋タスク3つが届く
2. **タスク記録** - 音声入力 → AI自動構造化（★最優先で議論中）
3. **返信溜まり防止** - Gmail未返信チェック＋LINEリマインド
4. **週次レビュー** - 溜まったタスクの棚卸し提案
